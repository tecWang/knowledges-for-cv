##### 反转链表

> 题目: 输入一个链表，反转链表后，输出新链表的表头。
> Input: {1,2,3,4,5}
> Output: {5,4,3,2,1}

```python
# -*- coding:utf-8 -*-
class Solution:
    def ReverseList(self, pHead):
        if pHead is None:
            return pHead
        
        last = None
        while pHead:
            temp = pHead.next
            pHead.next = last
            last = pHead
            pHead = temp
        
        return last # last会由尾节点逐步变为头节点
```
